DreAmy - Real-Time Image and Video Animation

Transforming real-time media into animations

This project utilizes Python Flask to create a web application that converts real-time images and videos into animations. The application captures live video feed from the user's webcam and applies various animation filters to transform the media into engaging animated visuals.

Documentation provided on Github

Technologies Used:

Python
Flask
OpenCV
JavaScript
HTML5
CSS3

How It Works:

Step One: Webcam Access

The application requests access to the user's webcam to capture video input.

Step Two: Image Processing

Using OpenCV, the application processes the video frames in real-time, applying filters and animations to create a dynamic visual experience.

Step Three: User Interface

The front-end is built with HTML5 and CSS3, providing an intuitive interface for users to interact with the application and view the animated results live.

Step Four: Final Output

Users can capture animated frames and download them as images or GIFs for sharing on social media platforms.
